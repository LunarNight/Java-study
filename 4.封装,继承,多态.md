##4.1封装

    class Person{
        private int age;//需要封装的属性
        public void setage(int age){ //通过成员函数进行设置
            this.age=age;
        }
        public void getage(){//通过成员函数获取
            System.out.println("年龄是" + age);
        }
    }
    public class l {
        public static void main(String arg[]){
            Person a = new Person();
            a.setage(10);
            a.getage();
        }
    }

封装的步骤:  
1.使用private修饰需要被封装的属性.  
2.提供公共的方法设置或者获取该私有的成员属性.
